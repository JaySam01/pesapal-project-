# pesapal-project-

This project is a simplified relational database management system implemented
from scratch in Python for the Pesapal Junior Developer Challenge 2026.

## Features
- SQL-like interface
- Interactive REPL
- CRUD operations
- Primary & unique key enforcement
- Hash-based indexing
- Nested-loop JOINs
- Persistent file storage
- Demo web application using the custom DB

## How to Run
```bash
python db/repl.py
python web_app/app.py
